# CBAM-PyTorch
Convolutional Block Attention Module implementation with PyTorch.
[Paper](https://arxiv.org/abs/1807.06521)

## Requirements
- Python 3.x
- PyTorch 1.4
- torchvision 0.5.0


## Quick Start
First, clone the code:

```
git clone https://github.com/JYPark09/CBAM-PyTorch.git
cd CBAM-PyTorch
```
